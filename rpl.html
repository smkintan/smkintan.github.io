Tentang Jurusan RPL

Sejarah Singkat: Jurusan Rekayasa Perangkat Lunak (RPL) di SMK Negeri 1 Indralaya Selatan berdiri untuk menjawab kebutuhan dunia industri digital terhadap tenaga profesional di bidang pengembangan aplikasi dan sistem informasi.

Visi:

Menjadi jurusan unggulan yang menghasilkan lulusan kreatif, inovatif, dan berkarakter unggul dalam bidang pengembangan perangkat lunak serta teknologi informasi.

Misi:

  • Mengembangkan pembelajaran berbasis proyek digital dan software development.
  • Menyiapkan siswa agar mampu membuat aplikasi desktop, web, dan mobile.
  • Membentuk karakter mandiri, profesional, dan kreatif di bidang teknologi.

Kompetensi Keahlian

  • Pemrograman Dasar
    Mempelajari dasar-dasar logika pemrograman dan algoritma dengan bahasa seperti C++, Python, atau Java.
  • Pengembangan Aplikasi Web
    Membuat website dinamis menggunakan HTML, CSS, JavaScript, PHP, dan database MySQL.
  • Pengembangan Aplikasi Mobile
    Mendesain dan membuat aplikasi Android menggunakan Android Studio dan Flutter.
  • Desain Antarmuka & UX
    Merancang tampilan aplikasi yang menarik dan mudah digunakan oleh pengguna.
  • Database & Sistem Informasi
    Mengelola database, membuat sistem CRUD, dan merancang aplikasi informasi sekolah, kasir, atau inventaris.

Fasilitas Laboratorium

  • Lab Pemrograman
    Dilengkapi dengan komputer berperforma tinggi untuk coding, debugging, dan testing aplikasi.
  • Lab Desain UI/UX
    Dilengkapi perangkat dan software desain seperti Figma, Adobe XD, dan Canva.
  • Server Development Room
    Tempat praktik pembuatan server, API, dan database cloud.
  • Internet & Jaringan
    Mendukung pembelajaran online, upload project ke GitHub, dan kolaborasi tim.

Peluang Karir Lulusan

  • Software Developer
    Membuat aplikasi desktop, web, dan mobile untuk kebutuhan bisnis atau publik.
  • Frontend / Backend Developer
    Spesialis dalam pengembangan tampilan dan sistem aplikasi berbasis server.
  • Database Administrator
    Mengelola dan merancang sistem database perusahaan.
  • Game Developer
    Mengembangkan game edukatif dan hiburan berbasis Android atau desktop.
  • UI/UX Designer
    Merancang tampilan aplikasi yang interaktif dan nyaman digunakan.
  • Freelancer / Startup Founder
    Mendirikan usaha digital mandiri di bidang software dan IT service.
